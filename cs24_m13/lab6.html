Lab 6: Sorting

Goals for this lab

By the time you have completed this lab, you should be able to sort a list in an ascending order.

Lab preparation

After logging in, create the directory for this lab:

mkdir -p cs24/lab6

Change into the lab2 directory and copy the files you will need for this lab:

cd cs24/lab6
cp ~bboe/public_html/cs24_m13/code/lab6/* .

Overview

This lab is a very small one. All you need to do is to write sort function (definition in sort.cpp) for the List class. You can implement any of the bubble, insertion or selection sort algorithms.
